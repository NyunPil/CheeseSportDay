# 🧀 치즈 운동회 — 드래프트 아레나 (VRChat 맵)

팀 드래프트(팀원 뽑기) 진행 전용 VRChat 룸. 정면 대형 스크린에서 드래프트가 진행되고,
팀장들은 앞열에, 팀원/관객은 **원형 경기장(스타디움)식 관중석**에서 다 같이 참여하는 구조.

> 이 폴더(`CheeseSportsArena`)를 **유니티 프로젝트의 `Assets/` 안에 통째로 복사**하면 됩니다.

---

## 빠른 시작 (3단계)

1. **VRChat 프로젝트 준비**
   - [VRChat Creator Companion(VCC)](https://vrchat.com/home/download) 으로 새 **Worlds** 프로젝트 생성
     (VCC가 알아서 호환되는 유니티 버전 + VRChat SDK3 - Worlds 를 세팅해줌)
   - ※ 직접 유니티만 켜서 시작했다면, VCC에서 SDK를 나중에 추가해도 됨

2. **이 폴더 복사**
   - `CheeseSportsArena` 폴더를 프로젝트의 `Assets/` 안에 드래그
   - 유니티가 자동으로 스크립트를 컴파일함 (콘솔에 에러 없으면 OK)

3. **맵 생성**
   - 상단 메뉴 **`Tools ▸ 🧀 치즈 운동회 ▸ 아레나 빌더 열기`**
   - 창에서 값 조정(팀 수 5↔4 등) 후 **`🧀 맵 생성`** 클릭 → 끝!
   - 다시 누르면 깨끗이 지우고 새로 만듭니다. (`맵 삭제`로 제거)

## 생성되는 것

| 요소 | 설명 |
|------|------|
| **MainScreen** | 정면 대형 드래프트 스크린(발광 디스플레이). 여기에 카드/현황을 띄움 |
| **DraftPanels** | 양옆 팀별 드래프트 현황 패널 4개 |
| **LeaderSeats** | 팀장 앞열(둥근 배치) — 팀이름 라벨 + **빨간 "뺏어오기" 버튼** 포함 |
| **AudienceStands** | 단(段)별로 올라가는 원형 경기장식 관중석 (감싸는 각도/단 수 조절 가능) |
| **OuterWall** | 바깥 원형 벽 + 입구 틈 |
| **Lighting** | 디렉셔널 + 스크린/무대 글로우 |
| **SpawnPoint** | 입장 위치(스크린을 바라봄) |

조절값: 팀 수, 팀당 인원, 앞열 반지름/각도, 관중석 단 수·감싸는 각도·반지름·높이, 스크린 크기 등.

## VRChat 기능 붙이기 (맵 생성 후)

빌더는 **순수 유니티 형태**까지만 만듭니다. 아래는 VRChat에서 동작시키기 위한 다음 작업:

### 1) 월드 기본
- 빈 GameObject에 **`VRC Scene Descriptor`** 추가 → `Spawns`에 `CheeseSportsArena/SpawnPoint` 연결
- 바닥/벽 등 밟고 부딪히는 오브젝트는 Collider 유지(프리미티브는 기본 포함됨)

### 2) 의자 앉기
- 각 좌석(`Seat_*`, `Leader_*`)에 **`VRC Station`** 컴포넌트 + 작은 트리거 콜라이더를 붙이면 앉기 가능
- 팁: 좌석 하나를 prefab으로 만들어 Station까지 세팅 후, 빌더가 만든 좌석을 prefab으로 교체하면 편함

### 3) 드래프트 기능 (Udon / UdonSharp)
드래프트가 메인이라 스크린 인터랙션이 핵심입니다. 권장 구현 순서:
- **스크린에 영상/이미지** : `VRC Unity Video Player` 또는 머티리얼 교체로 카드 표시
- **카드 뒤집기 + 등장 모션** : UdonSharp에서 애니메이션/트윈으로 카드 회전·확대
- **현황 패널 동기화** : `[UdonSynced]` 변수로 팀별 픽 상태 네트워크 동기화
- **뺏어오기 버튼** : `StealButton`에 Interact 이벤트 → 빨간 "잠깐만! 뺏어오기!" 오버레이 On, 같은 버튼 다시 누르면 Off (synced bool 토글)

> 마이크 증폭은 안 함 — 음성은 디스코드. 맵에는 아바타 오디오만.

### 4) 최적화 (정원 16명)
- 좌석/벽은 같은 머티리얼을 공유하므로 가벼움. 필요시 단 수/관중 좌석을 줄이세요.
- 라이트는 가능하면 Baked 권장. 관중석 좌석은 정적이면 **Static** 체크 후 라이트맵 베이크.
