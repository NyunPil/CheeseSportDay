## 1. 한 줄 요약
게임 운동회의 **팀 드래프트(팀원 뽑기) 생중계 무대**가 되는 원형 경기장 VRChat 룸.
정면 대형 스크린에서 드래프트가 진행되고, 팀장 5명이 앞열에, 팀원·관객이 뒤쪽
**원형 스타디움 관중석**에서 다 같이 지켜보는 구조. 동시 최대 **16명** 입장.

## 2. 콘셉트 & 무드
- **테마**: "치즈" + "운동회". 밝고 쫀득하고 귀여운 분위기 (참고: 로고 `64ffce0e3240007a.png`).
- **느낌**: 방송 예능 세트 + 미니 스타디움. 화사하고 들뜬, 축제 같은 톤.
- **핵심 포인트**: 드래프트가 메인 콘텐츠 → **정면 스크린이 시각적 주인공**. 나머지는 스크린으로 시선이 모이게 받쳐주는 역할.
- 치즈 모티프(구멍, 노란 질감), 운동회 모티프(만국기·트랙 라인·응원봉 등) 자유롭게 활용 환영.

## 3. 공간 구조 (위에서 본 배치)

```
            [ 입구 ]
   ┌──────────────────────────┐   ← 바깥 원형 벽
   │      [메인 스크린]         │     (정면, 가장 높고 큼)
   │  패널 패널 패널 │ 패널 패널 │   ← 현황패널 5개 (왼3 / 오2, 스크린 양옆)
   │                           │
   │     열무   …  v …   오늘   │   ← 팀장 앞열 (둥근 v자, 빨간버튼 정면)
   │        [ 중앙 무대 ]        │
   │   ◦ ◦ ◦ ◦ ◦ ◦ ◦ ◦ ◦       │   ← 관중석 1단
   │  ◦ ◦ ◦ ◦ ◦ ◦ ◦ ◦ ◦ ◦      │   ← 2단 (한 단 올라감)
   │ ◦ ◦ ◦ ◦ ◦ ◦ ◦ ◦ ◦ ◦ ◦     │   ← 3단, 4단 …
   │          [ 스폰 ]          │   ← 입장 위치 (스크린을 바라봄)
   └──────────────────────────┘
```

## 4. 요소별 상세 (역할 / 현재 상태 / 디자인 방향)

### ① 메인 스크린  ⭐가장 중요
- **역할**: 드래프트 카드 등장·뒤집기·팀별 현황이 뜨는 메인 화면. 모든 시선의 초점.
- **현재**: 어두운 프레임 + 발광 디스플레이(파란빛) 판때기 상태.
- **디자인 방향**: 방송 세트의 메인 LED월 느낌. 프레임을 치즈/운동회 모티프로 꾸며도 좋음. 디스플레이 면 자체는 영상/이미지가 들어갈 자리라 **가운데는 비워둘 것**.

### ② 현황패널 ×5 (왼쪽 3 · 오른쪽 2)
- **역할**: 팀별로 "누가 뽑혔는지" 로스터를 보여주는 보드. 팀당 1개.
- **현재**: 기둥 + 연한 발광 보드, 위에 팀 이름 라벨(열무/느루/호두/반부/오늘), 아래 빈 슬롯(`· ____` ×4).
- **디자인 방향**: 팀별 컬러/엠블럼으로 구분되면 좋음. 스크린을 가리지 않게 양옆에 배치돼 있으니 **위치는 유지**하고 외형만.

### ③ 팀장 좌석 ×5 (둥근 v자 앞열)
- **역할**: 팀장 전용석. 앞쪽 중앙(호두)이 살짝 들어가고 양 끝(열무·오늘)이 스크린에 가까운 **둥근 v(U)자**로, 무대를 감싸 안듯 앉음.
- **현재**: 금색 의자(일반석보다 큼) + 좌석마다 팀 이름 라벨.
- **디자인 방향**: "팀장"답게 일반 관중석보다 화려하게 (왕좌/캡틴석 느낌). 팀 컬러 포인트 환영.

### ④ 뺏어오기 빨간 버튼 ×5
- **역할**: 누르면 화면에 "잠깐만! 뺏어오기!" 빨간 연출이 뜨는 버튼. 다시 누르면 사라짐(기능은 추후 구현).
- **현재**: 각 팀장 **정면**에 받침대 + 빨간 원형 버튼 podium.
- **디자인 방향**: 눈에 확 띄는 위험-빨강. 게임쇼 버저/비상버튼 느낌. **위치(팀장 정면)는 유지**.

### ⑤ 관중석 (원형 스타디움)
- **역할**: 팀원·관객이 앉아 함께 지켜보는 곳. 뒤로 갈수록 한 단씩 올라가 다 보이게.
- **현재**: 4단, 단마다 받침(계단) + 크림색 의자가 호를 따라 쪼로록. 좌석 수는 단당 약 17~32석(여유분, 조절 가능).
- **디자인 방향**: 야구장/콘서트 객석 느낌. 단(계단) 재질, 좌석 색, 통로·난간 등 자유. **너무 무거워지지 않게** (아래 기술 제약 참고).

### ⑥ 중앙 무대 / 바닥 / 벽 / 입구
- **무대**: 중앙의 살짝 높은 원형 단. 운동회 트랙·치즈 원형 무늬 등 환영.
- **바닥**: 넓은 원형. 트랙 라인, 팀별 구역 색 등 가능.
- **벽 + 입구**: 바깥 원형 벽(높이 5.5 m)과 정면 입구 틈. 만국기·배너·치즈 구멍 등.

### ⑦ 조명 / 분위기
- **현재**: 기본 태양광 + 스크린 글로우 + 무대 스포트(임시).
- **디자인 방향**: 무대 조명 무드(스포트라이트, 색조명), 스카이박스/배경, 전체 톤 결정 환영.

## 6. 기술 제약 (VRChat 월드)
- **플랫폼**: VRChat. **Built-in 렌더 파이프라인** (URP/HDRP 아님). 셰이더는 Standard 호환 권장.
- **인원**: 동시 16명 → 가벼워야 함. 가능하면 **조명은 베이크(Baked)**, 정적 오브젝트는 Static + 라이트맵.
- **드로우콜/머티리얼**: 관중석 좌석처럼 반복되는 건 **머티리얼/메시 공유**로 최적화. 텍스처는 아틀라스 권장.
- **(선택) Quest 대응 시**: 폴리곤·텍스처 더 빡세게. PC 전용이면 여유 있음. — *PC 전용인지 Quest 겸용인지 확정 필요*.
- **스케일 기준**: 사람 키 ≈ 1.6~1.8 m. 의자/버튼/문은 이 기준에 맞춰져 있음.
